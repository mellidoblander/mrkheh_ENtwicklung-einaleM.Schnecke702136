Digitale Entwicklungsschnecke

Für jedes Kind kannst du eine eigene Schnecke speichern. Ein Klick auf eine Zahl färbt das Feld ein, ein zweiter Klick entfernt die Farbe wieder. Alle Daten werden direkt auf deinem Gerät gespeichert.

Tipp: Für eine schnelle Übersicht kannst du unten rechts zusätzlich Zahlen direkt antippen.
